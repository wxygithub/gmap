ExtLargeMapControl Example

To use the progress bar, create a ExtLargeMapControl object and add this control to the map wiith GMap2.addControl(ExtLargeMapControl).

var map = new GMap2(document.getElementById("map_canvas"));
map.setCenter(new GLatLng(37.4419, -122.1419), 13);
var extLargeMapControl = new ExtLargeMapControl();
map.addControl(extLargeMapControl);
