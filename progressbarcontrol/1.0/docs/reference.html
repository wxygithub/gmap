ProgressbarControl

Creates a progress bar control for usage in google maps. It can be used to show the progress of loading markers, for example.

For a description and examples of how to use this library, check out the how-to.

class ProgressbarControl

Constructor

Constructor Description
ProgressbarControl(Map:GMap2, opt_opts:ProgressbarOptions) Creates a progress bar control on the given map, with the given options.

Methods

Methods Return Value Description
remove() None Remove control.
start(operations:Number) None Start the progress bar.
updateLoader(step:Number) None Update the progress with specified number of operations.

class ProgressbarOptions

This class represents optional arguments to ProgressbarControl, There is no constructor for this class. Instead, this class is instantiated as a javascript object literal.

Properties

Properties Type Description
loadstring String Specifies the string displayed when first starting the control, before any update. The default value is Loading....
width Number Specifies, in pixels, the width of the bar. The default value is 176.
