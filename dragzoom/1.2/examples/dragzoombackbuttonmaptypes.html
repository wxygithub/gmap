Illustrating the restoring of map context including the map type using the DragZoom back button.
  • Select an area using the DragZoom around New York Harbor (about 1/3 of the screen). When the map has zoomed in, change the Map Type to Hybrid.
  • Do another DragZoom (don't click the back button yet) highlighting the Brooklyn waterfront and then change the Map Type to Satellite.
  • Now use the DragZoom back button and you will see the sequence reversed. The Map Types will be restored to the value for each zoom level.
